Weather Station using DHT11

Library Used:
DHT sensor library by Adafruit
Version: 1.4.6

Board:
Arduino Uno

Sensor:
DHT11

Outputs:
- Temperature (°C)
- Temperature (°F)
- Humidity
- CSV format on Serial Monitor
- Red LED for high temperature (>35°C) or humidity (>80%)
- Green LED for normal conditions

# Arduino UNO Pin Types

| Pin Type           |       Explanation                                                                                                                                             | **Example**                             | **Real IoT Use Case**                                                                       |
| ------------------ | ----------------------------------------------------------------------------------------------------------------------------------------------------------- | --------------------------------------- | ------------------------------------------------------------------------------------------- |
| Digital Input      | Reads only two states: HIGH (1) or LOW (0). Used to detect ON/OFF signals from switches or sensors.                                                         | Push button connected to Digital Pin 2. | Detecting whether a door is open or closed in a smart home security system.                 |
| **Digital Output** | Sends HIGH (5V) or LOW (0V) signals to control external devices.                                                                                            | LED connected to Digital Pin 13.        | Turning lights or relays ON/OFF in a home automation system.                                |
| **Analog Input**   | Reads varying voltage levels (0–5V) and converts them to digital values (0–1023). Available on pins A0–A5.                                                  | Potentiometer connected to A0.          | Monitoring temperature, light intensity, or soil moisture in smart agriculture.             |
| **PWM Output**     | Produces a Pulse Width Modulated signal to simulate analog output. Available on pins 3, 5, 6, 9, 10, and 11.                                                | LED brightness control using Pin 9.     | Controlling motor speed or adjusting smart lighting brightness.                             |
| **I2C / SPI Pins** | Communication pins used to connect multiple sensors and modules efficiently. **I2C:** SDA (A4), SCL (A5). **SPI:** SS (10), MOSI (11), MISO (12), SCK (13). | I2C OLED display or SPI SD card module. | Connecting multiple sensors or displays in weather monitoring and IoT data logging systems. |

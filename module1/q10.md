

A .gitignore file is a special file used by Git to specify which files and folders should **not** be tracked or uploaded to a Git repository. It is commonly used to ignore compiled files, temporary files, operating system files, and IDE configuration folders that are unnecessary for source code management.

### .gitignore for an Arduino Project

gitignore
# Compiled output files
*.hex
*.elf

# OS-specific files
.DS_Store
Thumbs.db

# IDE configuration folders
.vscode/
build/



# 1. git clone

* git clone creates a complete copy of a remote Git repository on your local computer.
* It downloads all files, branches, and commit history.
*  When you are working with a repository for the first time.

Example:

 https://github.com/username/iot-summer-school-2026.git



### 2. `git fetch`

* git fetch downloads the latest changes from the remote repository without modifying your local files or current branch.
* It allows you to review changes before merging them.
*  When you want to see the latest updates from collaborators without affecting your current work.

**Example:**


git fetch origin

### 3. `git pull`

* git pull downloads the latest changes from the remote repository and automatically merges them into your current branch.
* It is equivalent to running `git fetch` followed by `git merge`.
*  When you want to update your local repository with the latest changes and merge them immediately.

**Example:**


git pull origin main
